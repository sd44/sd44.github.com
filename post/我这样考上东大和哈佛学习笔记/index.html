《我这样考上东大和哈佛》学习笔记

发表于： 2018年03月08日 分类于： 学习方法
字数： 2362 阅读：≈ 5分钟 浏览： 评论：

给孩子做个学习方法《我这样考上东大和哈佛》的笔记。

16倍速学习法

学习成果 = 资质 x 策略 x 时间 x 效率

资质

资质就是头脑能力。

  • 相信自己的头脑很好。“这么好的头脑不好好发挥，岂不是太浪费了！我要把潜藏在脑中、不为人知的神秘力量激发出来。我要召唤脑中那群优秀、强大、蓄势待发的精英部队。”
  • 阅读能力是一切的基础。
  • 写作能力。
  • 算数。相当于计算能力、数字创造力、推理能力等。
  • 随身带着书，随时看。
  • 选择读哪本书。花4、5分钟看看书名、封面文字、作者简介还有 目录 。高中或大学时和之后，看看书有哪些参考文献，以便选择下一本该读的书。
  • 在书中有印象的地方，或者重要的地方，画线和做笔记。
  • 不要一下子就看很难的专业书籍，可按漫画书->小说->入门书->专业书的顺序来，由易到难。
  • 当读书是为了某个明确的目的时，就可以用速读来掌握重点。这通常是在读实用书籍时会用的方式。阅读实用书籍的第二个秘诀是，只要读到重点，务必确定具体范例和实践步骤1。
  • 好书要读出声音，要反复阅读。
  • 经常结合书本思考、标记，
  • 做笔记，写读后感或总结。不一定非要是文字，也可以是关键词、图画、线条等笔记符号。
  • 写作时（包括读后感、总结或书摘），一定要定出时间限制和目标字数。考虑记叙文六要素5W1H（时间、地点、人物、起因、经过、结果）。
  • 思考现实中的各种数字问题，就会看懂以往含糊不清的东西，甚至想出新的创意来。思考当前的复杂情况是否可以套用小学文化程度的数学就能解决。

策略

针对学习的目的、目标，彻底思考应该采取何种学习方式。 能够以一年、数个月为单位提高学习成果的方法，就是制订“策略”。

自己动脑思考 做什么、何时做、做多少以及怎么做，自然而然就会提高学习成果。

  • 想要成功学习，就必须树立明确的目标，从心底强烈渴求“我要做”才行。为此，目标绝对不能模糊，一定要非常具体。
  • 设定达到目标的期限。一定要限定“到何时为止”的目标达到期限才行。告诫自己“这天之前没达到就惨了”。
  • 充分了解你的目标是怎么构成的？有哪些特征？最近趋势如何？（细分）
    • 横向分解目标的要素：如具体有哪些科目，每科目具体分数。然后再把每一科目细分，如选择题目标分数多少，填空题目标分数多少，应用题目标分数多少。不停细分，直到觉得自己做得到为止。
    • 纵向分解目标的时间轴：例子，目标：6个月后会写600个字。达到目标所需的进度：每月至少会写100个字。纵向分解：一个月后，学习每10个字需要20分钟时间；二个月后，每10个字需要18分钟时间；三个月后，只需15分钟……持续细分，把一年后，半年后的目标，逐步分解成“一个月后的目标”，“本周目标”，“每天目标”
  • 询问这方面的专家。有人会认为请教次数过于频繁，会给对方带来麻烦。但是与其烦恼这种事情，还不如加强“无论如何都要达到目标”的信念。这里要注意的是，询问专家时，必须向对方完整提供“自己目前正属于何种状态”的信息。
  • 用历年真题、模拟考来了解自己目前的水平。
  • 可先从简单的科目或者擅长的项目做起。先易后难。打下一定基础后，接下来尽快攻克困难的或者不擅长的领域。因为能带来最大增长的，一定是你不擅长的领域。
  • 向他人宣布你的目标以维持决心。外人会给你学习的压力和动力，也会给你鼓励和支持。
  • 想象成功的瞬间与达到目标后的景象。

时间

学习时间就相当于士兵数量，数量越多，胜利几率越高。 增加学习时间是非常实在、而且无法避免的一条路。 学习最后还是得用最脚踏实地的努力与毅力来分胜负。

  • 人一天可以学习14个小时。
  • 1小时 x 一个月的学习量 = 10小时 x 三天的学习量。
  • 增加学习时间，是学习起步阶段的主要目标。可以慢慢增加学习时间，让身体习惯，让自己觉得“我应该可以再多学会儿”。
  • 学习时间分割成小块，如这一小时学什么，下一小时学什么。
  • 坐车时，等车时，等人时也是学习时间，可看一些易携带的小书。
  • 在厕所或房间墙壁上贴上重点知识。
  • 按科目重要程度来安排时间。如数学比手工重要。数学可以学习2小时，手工可以学习半小时。
  • 减少时间浪费，不要花太多时间整理笔记或买太多类似参考书。

效率

效率每单位时间的学习效果。

  • 专心做一件事，不要一心二用。
  • 学习的核心就是解出考题以解出考题为学习重心。
  • 不要害怕面对分数。分数不是别人用来评判自己的东西，而是用来了解自己的理解程度和熟练程度，用来告诉自己距离自己目标还有多远。但是不能让心情随着分数的高低而起伏，要冷静而深入地掌握自己目前的学习情况。
  • 重视解答之后的反思时间，以及选择解析较详细的题库的搭配方案。答错下次就必须再解一次。
  • 用印象与关键词来抓重点的默背法，并反复练习。如根据课文内容结合动画情节等。
  • 不要死背公式，要用逻辑推导，用逻辑的方法理解公式为什么会成立。用逻辑的方法思考问题，可说是相当费时又累人的工作，但是从结果来看，却是最有效率的学习法。
  • 参考书上要画线及作记号，还有随手做笔记。
  • 准备一个好的学习环境。首先，请保持桌面清洁。接下来，把该做的题目和参考书、字典放在书架上显眼的地方。准备需要用到的笔、纸、橡皮和相关工具。可以在墙上贴上目标、名人名言和时间分配表等。
  • 消除烦恼，提高注意力。
  • 学习之前，要设定目标和限制时间。
  • 要能自由运用集中注意力的开关。
  • 以玩乐感、好奇心，或输赢感享受学习。
  • 感觉每天的学习都像上考场。花5秒到1分钟时间，闭上眼睛假设自己正在参加考试，记下开始答题时的时间。
  • 考场上，最集中的注意力可以决定胜负（临场发挥能力）。

1 不能只是看看重点就以为自己懂了。了解重点只是达到知识的阶段，这与实践（具体应用重点）的阶段还有一段距离。另外，每本书的特色就在于作者以自身经验或实际采访所得到的范例。具体范例即包括实践性的提示，也比较容易套用在自己身上，留下深刻印象。

确认重点之后，再看看自己有没有经历过书中的具体案例。不要因为读过就满足了，一定要把书中的知识结合自己的生活加以实践才行。
