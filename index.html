告别自卑，超越自我：阿德勒《自卑与超越》

发表于： 2025年08月03日 分类于： 心理学 浏览：

在追求内心平静与自我成长的旅途中，我们常常会遇到一个无形的障碍：自卑。它像一张无形的网，束缚着我们的潜能，阻碍我们迈向更广阔的世界。作为心理学界的一部里程碑式作品，阿尔弗雷德·阿德勒的经典著作《自卑与超越》深刻剖析了自卑情结的根源及其对个体发展的影响。阿德勒，作为个体心理学的创始人，他将心理学从纯粹关注个人本能和潜意识，拓展到强调个体与社会环境的互动，以及个人对意义和归属的追求，为我们揭示了如何将这份“不足感”转化为积极向上的动力，最终实现自我超越。

直面自卑，探寻根源

阿德勒认为，人类普遍存在自卑感，许多人际关系中的问题，都与每个人内心深处的自卑和对超越的渴望 有关。这种感觉并非消极的弱点，而是驱动个体（乃至社会）成长和进步的内在动力。不管想与不想，每人都活在“意义之中”。每个人内心深处都或多或少藏着一份自卑感，这很正常。它可能源于我们觉得自己不够好、不够美，或者某些方面不如别人。这本书会帮助您理解这份感觉从何而来，无论是小时候的经历，还是平时与他人的比较，都能在书中找到共鸣和答案，让您更了解自己。

发表于： 2018年09月19日 分类于： 社科 浏览：

本文说明：本文是我的《梦园呓语——我的三观独白》中「空间生产·济南市丁家庄城中村见闻散记」章节的草稿。完整书籍 PDF 及在线阅读请访问： 我的书《梦园呓语——我的三观独白》 。

以下只是粗陋的第一版草稿，可直接移步 Github 下载最终成书 PDF，获得更好观感。

背景介绍

济南市丁家庄，又名丁家村、丁家新村，据1992年5月1日所立村碑记载：

明永乐年间（1403-1424）当地根据传说取村名"定妖庄"。后因此名不雅，故以"定"字谐音"丁"字改为丁家庄。

丁家庄隶属于山东省济南市姚家街道，是济南市一个较大且密集的城中村，在二十年前就已开始为外来务工人员提供住房等服务，共有村民宅基地（院落）近800户5000人，外来流动人口峰值大约可达30000人。丁家村城中改造是山东省棚改旧改的重点项目，于2017年年底基本完成房屋拆除工作，拆迁面积约为53万平方米（村民宅基地、公益性公共设施用地加上经营性用地等）。

空间生产之曾经的城中村的基本概念

发表于： 2018年09月16日 分类于： 城中村 浏览：

本文说明：本文是我的《梦园呓语——我的三观独白》中「空间生产」章节的相关内容。完整书籍 PDF 及在线阅读请访问： 我的书《梦园呓语——我的三观独白》 。

空间生产之曾经的城中村的基本概念

所谓城中村者，城市和村庄性质兼而有之：它深处城市之中，作为城市的一部分，周边均具城市特征，自身却充斥着农村式的无序和自然，缺乏人工的总体规划，各家各户的宅地界限比纯粹的村庄还要不清晰和混乱，基础设施（能源、通讯、供水、交通、安全、卫生、医疗、文化等）薄弱，原生住户基本为农村户籍，土地制度仍为农村集体所有制而非城市的全民所有制；作为农村，它的外来流动人口数量数倍，甚至数十倍于原生居民，耕地被大量或完全占用，转为商业或住宅地产，耕地的这种性质转变使原生居民原先赖以生存的农业收入转为地产收入，并成为原生居民收入的重要来源。
